A very simple API.

## Installing / Getting started

I used `Yarn` for this project. It is set to listen to `localhost:3333`.

```shell
$ yarn
$ yarn dev
```

Will install dependencies and run project.

## Developing

### Built With
> Express 4.17.1
> Nodemon 2.0.6
> Sucrase 3.16.0

```shell
git clone https://github.com/leobezr/CRUD-backend.git
cd your-project/
yarn
```
